# Electron Svelte Store Example

Use electron-store in 2 way:

1. Traditional Svelte Stores
2. Svelte5 Rune

# Technology Stack

- Frontend:
  - Svelte 5
  - TypeScript
  - Vite (via electron-vite)
- State Management:
  - Traditional Svelte Stores
  - Svelte5 Rune
- Build Tools:
  - electron-vite
  - TypeScript

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```
